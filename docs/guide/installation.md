# 安装

## 包管理器安装

### npm

```bash
npm install moonwind-ui
```

### yarn

```bash
yarn add moonwind-ui
```

### pnpm

```bash
pnpm add moonwind-ui
```

## CDN 引入

你可以通过 CDN 的方式引入 Moonwind UI，适合快速原型开发。

```html
<!-- 引入样式 -->
<link rel="stylesheet" href="https://unpkg.com/moonwind-ui/dist/style.css" />

<!-- 引入 Vue 3 -->
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

<!-- 引入 Moonwind UI -->
<script src="https://unpkg.com/moonwind-ui/dist/moonwind-ui.umd.cjs"></script>
```

## 版本要求

Moonwind UI 需要以下环境：

- Vue 3.3.0+
- Node.js 16+
